## steps:

- The pipeline uses orbs to install Node, the AWS cli and the EB cli.

- It checks out the code from the repo
- FrontEnd & BackEnd install
- FrontEnd & BackEnd build
- Wait for admin approval
- FrontEnd & BackEnd deploy
